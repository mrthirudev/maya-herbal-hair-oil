# MAYA Herbal Hair Oil — React + TypeScript

This is a small demo web app in Tamil for selling MAYA Herbal Hair Oil.

Quick setup (using yarn):

```bash
yarn install
yarn dev
```

Assets:
-- Place the provided images into `public/assets/` as either PNG/JPG or SVG. Recommended names:
  - `logo.png` or `logo.svg` — brand logo (from attachments)
  - `banner.png` or `banner.svg` — product/banner image (from attachments)

The project includes simple SVG placeholders at `public/assets/logo.svg` and `public/assets/banner.svg`. If you add `logo.png` / `banner.png`, they will replace the placeholders automatically.

The app uses Vite + React + TypeScript. The main source files are in `src/`.
